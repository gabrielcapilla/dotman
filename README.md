# dotman

A high-performance dotfiles manager written in Nim with multi-profile support and automatic conflict detection.

## Documentation

Full documentation: https://gabrielcapilla.github.io/dotman/docs.html

## Installation

```bash
curl -sL gabrielcapilla.github.io/install | sh -s dotman
```

Or via Nimble (requires Nim 2.2.6+):

```bash
nimble install https://github.com/gabrielcapilla/dotman.git@#head
```
